Slider

Slider 컴포넌트는 사용자가 특정 범위 내에서 값을 선택할 수 있는 직관적인 인터페이스를 제공합니다.

특징​

  • 가로 및 세로 방향 지원
  • 다양한 크기와 색상 변형
  • 커스텀 가능한 범위와 스텝
  • 접근성 고려
  • 제어 및 비제어 모드 지원
  • 다크 테마 지원

사용법​

import { Slider } from 'react-common-components-library';

function SimpleSlider() {
  return <Slider defaultValue={30} />;
}

API 참조​

속성타입기본값설명
valuenumber-제어 컴포넌트로 사용할 때의 값
defaultValuenumber0초기 값
minnumber0최소값
maxnumber100최대값
stepnumber1증감 단위
disabledbooleanfalse비활성화 여부
orientation'horizontal' | 'vertical''horizontal'슬라이더 방향
size'sm' | 'md' | 'lg''md'슬라이더 크기
showValueLabelbooleanfalse값 라벨 표시 여부
formatLabel(value: number) => string(val) => String(val)값 라벨 포맷 함수
onChange(value: number) => void-값 변경 시 호출되는 콜백
onChangeComplete(value: number) => void-슬라이더 조작 완료 시 호출되는 콜백
classNamestring-추가 CSS 클래스
styleReact.CSSProperties-인라인 스타일
namestring-폼에서 사용할 이름
idstring-컴포넌트의 ID

접근성​

Slider 컴포넌트는 WAI-ARIA 디자인 패턴을 준수하여 구현되었습니다:

  • role="slider" 속성을 사용합니다
  • aria-valuemin, aria-valuemax, aria-valuenow 속성을 제공합니다
  • aria-disabled를 통해 비활성화 상태를 제공합니다
  • aria-orientation을 통해 방향을 제공합니다

키보드 탐색​

  • ← / ↓ : 값을 1단계 감소
  • → / ↑ : 값을 1단계 증가
  • PageDown : 값을 10단계 감소
  • PageUp : 값을 10단계 증가
  • Home : 최소값으로 이동
  • End : 최대값으로 이동

TypeScript 지원​

Slider 컴포넌트는 TypeScript로 작성되었으며 타입 정의를 제공합니다:

import { Slider } from 'react-common-components-library';

function TypedSlider() {
  const handleChange = (value: number) => {
    console.log(`현재 값: ${value}`);
  };

  return (
    <Slider
      defaultValue={50}
      min={0}
      max={100}
      step={5}
      onChange={handleChange}
    />
  );
}
